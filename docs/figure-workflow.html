图表生成与同步CSV 到论文图的可复现路径
首页图表模板

图表也是实验产物，不是临时截图

图表脚本应固定输入、标签映射、字体、输出目录和 LaTeX 同步策略。最终验收看 PDF 中的实际页面。

目录

脚本规范

  • 显式声明输入 CSV、输出目录和 LaTeX 图目录。
  • 集中维护中文标签、模型名映射、方法顺序和色板。
  • 设置中文字体候选，关闭 unicode minus 问题。
  • 同时输出高分辨率 PNG 和 PDF；matplotlib 图保留矢量 PDF。
  • 生成 summary 文件，记录聚合结果和样本数。

视觉验收

检查项标准
文字中文不乱码，字号适合论文页面，标签不重叠。
图例方法顺序稳定，颜色含义一致。
指标图中数值与 CSV / summary 可追溯。
版式插入 LaTeX 后没有越界、压缩、浮动错位。

同步策略

建议使用单独 sync 脚本，把正式图清单从实验输出目录复制到 LaTeX 图目录。这样可以避免临时预览图、旧图和正式图混在一起。
