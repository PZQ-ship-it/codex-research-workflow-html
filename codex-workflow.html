使用本仓库时的默认工作顺序

先理解协作契约和事实源，再进入具体实验、图表或论文审阅任务。不要绕过事实源直接改正文结论。

推荐顺序

  1. 阅读 Codex 协作约定。
  2. 确认 事实源与同步纪律。
  3. 按任务选择实验、图表、审阅或 HTML 笔记模板。
  4. 若要让 Codex 自动进入工作流，复制相应 skills/* 到 $CODEX_HOME/skills 并重启 Codex。

边界

  • 不把未验证实验结果写成定论。
  • 不把 LLM 生成内容当作事实候选来源。
  • 不复制 API key、私有日志、大型 PDF 渲染产物到本模板仓库。
  • 所有可复用记录优先使用 HTML；Codex skill 自身保留必需的 SKILL.md。
