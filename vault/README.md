# Knowledge Vault

这是本项目的纯文本长期知识库，用于保存可跨会话复用的上下文、决策、Codex 使用技巧和项目状态。

## 目录

- `TODO.md`：待办、后续问题、需要验证的点。
- `agent/`：Codex 相关偏好、提示词模式、工具和工作流笔记。
- `projects/`：项目状态、里程碑、决策记录。
- `people/`：协作者/利益相关者笔记，避免敏感信息。
- `notes/`：通用知识笔记。
- `sources/`：资料来源、阅读路径、引用清单。
- `templates/`：可复制模板。

## 使用原则

- 先短后长：先保存可复用事实和下一步，再逐渐整理。
- 先索引后展开：重要主题可以先在 `INDEX.md` 留入口。
- 不放秘密：不要保存 `.env`、token、账号、私密数据。
- 版本敏感内容要标注来源或待确认。

## 建议入口

- 当前 Codex 高级用法主文档：`../docs/codex-advanced-config-and-usage.md`
- Vault 索引：`INDEX.md`
- 当前待办：`TODO.md`
