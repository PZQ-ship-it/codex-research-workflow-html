Adversarial QA Scenarios

  • Code: empty input, duplicate input, stale cache, partial failure, retry path, concurrent run.
  • Paper: inconsistent term, unsupported claim, metric name drift, figure/table mismatch, missing citation.
  • Figure: missing category, long label, font fallback, color collision, mobile/export scaling.
  • Benchmark: API timeout, malformed response, interrupted shard, model switch, cost boundary.
  • Prompt: ambiguous role, hidden assumption, over-broad scope, no evidence requirement.
