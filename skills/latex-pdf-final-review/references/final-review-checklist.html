LaTeX PDF Final Review Checklist

  1. Compile with the project standard command.
  2. Inspect errors, references, citations, missing assets, and severe layout warnings.
  3. Extract PDF text and search for stale terms, placeholders, old metrics, and inconsistent names.
  4. Render key pages: cover, abstract, TOC, main diagrams, experiment tables, conclusion.
  5. Compare figure/table numbers against source summaries.
  6. Separate human confirmations from automatic checks.
