Thesis Figure Checklist

  1. Confirm source table version and metric definitions.
  2. Use stable label maps, order arrays, and palettes.
  3. Configure CJK fonts and unicode minus handling.
  4. Generate PNG and PDF with deterministic filenames.
  5. Write summary tables with sample counts.
  6. Sync only final figures into the LaTeX figure directory.
  7. Render the final PDF page and inspect labels, legends, clipping, and readability.
