Deep Interview Pattern

Use this reference when a request needs clarification before planning or execution.

Question Style

  • Ask one question at a time.
  • Ask only questions that change scope, risk, design, or acceptance.
  • Prefer concrete options with tradeoffs over open-ended prompts.
  • Do not ask for facts that can be discovered from the repo or official docs.
